Descrição do Projeto

Este projeto foi desenvolvido para a disciplina de SOP com o objetivo de criar um site sobre a Copa do Mundo de 2026 utilizando HTML, CSS, Git e GitHub.

Estrutura do Site
Página Inicial

Apresenta informações gerais sobre a Copa do Mundo de 2026 e permite a navegação para as páginas dos países-sede.

Estados Unidos

Contém informações sobre os Estados Unidos, cidades-sede, cultura e curiosidades relacionadas à Copa do Mundo.

Canadá

Contém informações sobre o Canadá, cidades-sede, cultura e curiosidades relacionadas à Copa do Mundo.

México

Contém informações sobre o México, cidades-sede, cultura e curiosidades relacionadas à Copa do Mundo.
